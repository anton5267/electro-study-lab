# Roadmap

Поточний фокус: перетворити Electro Study Lab з навчального MVP у практичний тренажер для регулярної підготовки.

## Q1 2026 (березень-квітень)

- Розширити базовий контент до мінімум:
  - `quizData`: 80+ питань на мову.
  - `practiceProblems`: 40+ задач на мову.
  - `flashcards`: 60+ карток на мову.
- Додати контент-гайд (`CONTRIBUTING-CONTENT.md`) з правилами стилю, складності та валідації.
- Підготувати перший tagged release (`v1.0.1`) із зафіксованим changelog.

## Q2 2026 (травень-червень)

- Додати адаптивний режим тренувань:
  - підбір питань із `reviewQueue`;
  - пріоритезація тем з найнижчим mastery;
  - короткі мікро-сесії (5-10 питань).
- Додати інтервальне повторення флешкарток (spaced repetition light).
- Розширити browser smoke-тести для нових сценаріїв адаптивного навчання.

## Q3 2026 (липень-вересень)

- Зменшити розмір і складність `assets/js/app.js`:
  - винести секційні контролери (`quiz`, `exam`, `flashcards`, `import`) у окремі модулі;
  - зберегти `app.js` як orchestration layer.
- Вирівняти тестове покриття під нову модульну структуру.
- Додати performance baseline для cold start і first interaction.

## Q4 2026 (жовтень-грудень)

- Поглибити PWA/offline модель:
  - стратегія кешу для navigation/asset запитів;
  - стабільний offline fallback без побічних ефектів для API/resource-запитів.
- Додати локалізовані release notes (`UA/DE`) для публічних релізів.
- Підготувати `v1.1.0` як перший функціональний milestone.
